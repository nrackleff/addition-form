INTRODUCTION
------------
The Addition Form module was built to demonstrate creation of a Drupal 7 module that:
 * Creates a menu item that triggers a page callback.
 * Implements a form with two fields to enter numbers and a submit button.
 * Validates that the values entered into the two fields are numeric.
 * Displays the sum of the two numeric inputs in a standard Drupal message.


REQUIREMENTS
------------
No dependencies. 


INSTALLATION
------------
Install as you would normally install a contributed Drupal module. See:
https://drupal.org/documentation/install/modules-themes/modules-7 for further information.


CONFIGURATION
-------------
This module has no configuration settings.
